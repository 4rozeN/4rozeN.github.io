在Windows中启用DOH功能

本文最后更新于：2024年10月26日 晚上

Windows设置中启用DOH

适用环境：Windows11（Win10可以开启浏览器的DOH）
DNS检测工具：网易DNS在线检测
参考文章：

  • 网络设置DOH：Windows 11 22H2开启 DoH（DNS over HTTPS）
  • 检测：DNS、DoT、DoH协议测试方法

Google浏览器和Firefox浏览器启用DOH

Google浏览器版本：123.0.6312.86（正式版本） （64 位）
Firefox浏览器版本：123.0.1 (64 位)

Google浏览器

找到设置->隐私和安全->安全->高级在高级区启用使用安全DNS，并填写DNS提供商即可。

DNS提供商填写网络设置DOH这篇参考文章中的就好。
所有设置妥当之后请调用cmd输入ipconfig /flushdns该命令使得设置生效。

Firefox浏览器

在设置页面直接搜索dns即可。设置如图：

DNS提供商填写网络设置DOH这篇参考文章中的就好。
所有设置妥当之后请调用cmd输入ipconfig /flushdns该命令使得设置生效。

检测是否成功开启DOH

涉及工具：dog工具
协议：
-U, --udp Use the DNS protocol over UDP
-T, --tcp Use the DNS protocol over TCP
-S, --tls Use the DNS-over-TLS protocol
-H, --https Use the DNS-over-HTTPS protocol

命令示例：dog.exe -H @[https://dns.alidns.com/dns-query](https://dns.alidns.com/dns-query) bilibili.com
